# 🗺️ Zelda GeoGuessr

> Браузерная игра в стиле GeoGuessr по миру *The Legend of Zelda: Breath of the Wild*.  
> Угадайте место на карте Хайрула по скриншоту из игры!

## 🎮 Как играть

1. **Выберите серию** — каждая содержит 5 раундов разной сложности.
2. **Рассмотрите скриншот** — можно увеличить, перемещать и свернуть.
3. **Поставьте маркер** на карте, кликнув в предполагаемое место.
4. **Нажмите «Подтвердить»** — увидите расстояние до правильной точки и очки.
5. **После 5 раундов** — итоговый результат.

### Система очков

| Точность        | Очки         |
|-----------------|--------------|
| Идеально (0 px) | 5 000        |
| ~500 px          | ~3 900       |
| ~1 000 px        | ~3 000       |
| ~2 000 px        | ~1 840       |
| ~4 000 px        | ~675         |
| Далеко           | → 0          |

Максимум за серию — **25 000 очков**.

Формула: `S = 5000 × e^(−d / 2000)`, где `d` — расстояние в пикселях карты.

## 🚀 Запуск

### Простой способ (без сервера)

Откройте `index.html` в любом современном браузере. Всё работает локально.

### Через локальный сервер (опционально)

```bash
# Python 3
python -m http.server 8000

# Node.js (npx)
npx serve .
```

Затем откройте `http://localhost:8000`.

## ➕ Добавление своих локаций

### 1. Подготовьте скриншоты

Сделайте скриншоты в игре. Сохраните в папку `locs/` с именами вида:

```
location_s{номер_серии}_i{номер_раунда}.png
```

### 2. Определите координаты

Откройте карту `map/map_high.jpg` в графическом редакторе и запишите координаты (x, y) каждой локации в пикселях. Начало координат — левый верхний угол, X вправо, Y вниз.

### 3. Добавьте серию в `locations_data.js`

```javascript
{
    "id": 4,
    "name": "Моя серия: Название",
    "rounds": [
        { "image": "location_s04_i01.png", "x": 1234, "y": 5678 },
        { "image": "location_s04_i02.png", "x": 9876, "y": 5432 },
        { "image": "location_s04_i03.png", "x": 4500, "y": 7800 },
        { "image": "location_s04_i04.png", "x": 11000, "y": 3200 },
        { "image": "location_s04_i05.png", "x": 6000, "y": 12000 }
    ]
}
```

### 4. Готово

Перезагрузите страницу — новая серия появится в меню.

## 🗺️ О карте

Игра использует карту Хайрула размером **18 000 × 15 000** пикселей в пиксельной системе координат. Для отображения используется `L.imageOverlay` из Leaflet с `CRS.Simple`.

Рекомендуемое разрешение файла `map/map_high.jpg`:

| Разрешение   | Размер файла | Качество при приближении |
|-------------|-------------|-------------------------|
| 4 500 × 3 750 | ~3 МБ | Среднее |
| **9 000 × 7 500** | **~10 МБ** | **Хорошее (рекомендуется)** |
| 12 000 × 10 000 | ~18 МБ | Отличное |
| 18 000 × 15 000 | ~25 МБ | Максимальное |
